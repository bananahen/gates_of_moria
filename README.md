# The Gates Of Moria

A simple quiz to test Ruby knowledge.

## Installation

Add this line to your application's Gemfile:

    gem 'gates_of_moria'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gates_of_moria

## Usage

Run `gates_of_moria` and follow instructions.
